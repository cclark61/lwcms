      Data Query Class

      Pass the constructor an associative array:

      [type] -> 'insert', 'update', 'delete', 'select' [table] -> "table name" [fields] -> An associative array of fields and their values. $fields["field_name"] = ["field_value"] [filter_phrase] -> The full SQL "where" phrase used to filter results [order] -> order by phrase [group] -> group by phrase [limit] -> limit phrase
